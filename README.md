## 技术栈

- **开发语言**：Java
- **框架**：Springboot/SSM + Vue
- **JDK版本**：JDK 1.8（或11）
- **服务器**：Tomcat
- **数据库**：MySQL 5.7（或8.0）
- **数据库工具**：Navicat
- **开发软件**：Eclipse/IntelliJ IDEA
- **依赖管理包**：Maven

- 用户管理

### 用户角色

## 系统运行环境

- 服务器：Tomcat 8.x 或以上版本
- 数据库：MySQL 5.7/8.0
- JDK版本：1.8 或 11

## 目录结构

```
SpringbootMiaoshaSystem/
├── src/
│   ├── main/
│   │   ├── java/
│   │   └── resources/
│   └── test/
└── web/

    └── frontend/ (前台用户页面)
```

## 部署步骤

1. 确保Java开发环境（JDK 1.8或11）已安装。
2. 安装并配置MySQL数据库版本5.7或8.0。
3. 安装Tomcat服务器。
4. 使用Maven进行项目依赖管理。
5. 将项目部署至Tomcat服务器。
